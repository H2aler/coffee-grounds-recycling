# 커피박 재활용 웹페이지 ☕

최신 기술을 활용한 인터랙티브 커피박 재활용 정보 웹페이지입니다.

## 🚀 기술 스택

- **Next.js 15** - React 프레임워크
- **React 19** - 최신 React 버전
- **TypeScript** - 타입 안정성
- **Framer Motion** - 고급 애니메이션
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **Three.js** - 3D 그래픽 (준비됨)
- **Canvas API** - 파티클 효과

## ✨ 주요 기능

- 🎨 인터랙티브 파티클 배경 효과
- 📊 실시간 통계 애니메이션
- 🎭 부드러운 스크롤 애니메이션
- 🌓 다크모드 지원
- 🔍 실시간 검색 기능
- 📱 완전 반응형 디자인
- ⚡ 최적화된 성능

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 🎯 주요 컴포넌트

- `CoffeeParticleScene` - Canvas 기반 파티클 배경
- `InteractiveStats` - 애니메이션 통계 카드
- `FloatingCards` - 호버 효과가 있는 카드 그리드

## 🌈 디자인 특징

- 그라데이션 배경
- 글래스모피즘 효과
- 부드러운 전환 애니메이션
- 커피 테마 색상 팔레트
